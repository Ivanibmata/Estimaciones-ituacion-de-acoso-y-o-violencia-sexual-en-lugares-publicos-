# Estimaciones de situaciones de acoso

Repositorio que contiene las bases provenientes de la Encuesta Nacional de Seguridad Pública Urbana (ENSU) del segundo trimestre de 2025, así como el código utilizado para calcular las estimaciones al 90% de confianza.

## Contenido del repositorio

Bases de datos originales o procesadas de la ENSU 2025-T2.


Código en R para limpiar, procesar y generar estimaciones.


### Descripción general

Este proyecto tiene como objetivo estimar la prevalencia y características de situaciones de acoso reportadas en la ENSU (2025-T2). Incluye:

Procesamiento y limpieza de las bases.

Cálculo de indicadores y estimaciones con intervalos al 90% de confianza.
